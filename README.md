# ZGNBCARD使用说明
抽牌：点击卡牌本身或【抽牌】按钮，卡牌 3D 翻转随机显示语录，重复点击可换新；

触控解锁：点【🚫】按钮变【✋】，卡牌可点击抽牌，复点恢复锁定（仅按钮可抽）；

重置：点击【重置】，卡牌翻回背面，恢复初始 “点击抽取语录卡牌” 状态；

验牌：点【验牌】弹出所有语录列表，点语录左侧【🔍】可将该语录显示在卡牌并关弹窗，点【关闭】/ 弹窗外也可关闭列表；

小提示：操作提示悬浮 2 秒自动消失，页面适配手机等小屏设备。
